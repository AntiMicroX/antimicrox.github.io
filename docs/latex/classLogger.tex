\doxysubsection{Detailed Description}
Custom singleton class used for logging across application. 

It manages log-\/levels, formatting, printing logs and saving them to file. Logs across the program can be written using q\+Debug(), q\+Info(), q\+Warning(), q\+Critical, and q\+Fatal() functions
